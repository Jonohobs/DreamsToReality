## The Ralph Method

1. **Ask a focused question** to the AI
2. **Get the response** and extract key info
3. **Kill the context** (start fresh session)
4. **Route to someone else** or a different model for verification
5. **Iterate** until confident

**Why it works**: Prevents context pollution, gets fresh perspectives, catches hallucinations.

## GitHub Repository Analysis

### Recursive Star Search
```
1. Find relevant repos
2. Sort by stars (more stars = more validated)
3. Analyze top repos
4. Check what those repos star
5. Recurse through the network
```

### Colosseum/Hackathon Mining
- Search X/Twitter for project announcements
- Extract project repos from hackathon submissions
- Analyze winning patterns
